Branch & Pull Request (Web-Only)

  1. Upload files → choose Create a new branch → name it feature/lease-minimal-owner-inputs → Propose changes.
  2. Click Compare & pull request → Create pull request.
  3. Optional: Settings → Pages → temporarily publish from your new branch to preview.
  4. Happy? Merge pull request. If Pages is on main, the site redeploys.
